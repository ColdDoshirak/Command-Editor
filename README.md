# Command-Editor QUEUE BETA

Group Settings автоматически подтягивает группы из команд. Для музыки рекомендуется группа SONG и поменять аудио канал.
Новые команды:
- !top - выводит топ-5 людей по баллам.
- !give @username amount - передача баллов пользователю.
- !uptime - сколько длится стрим, и живой ли он.
Команды выше были добавлены нейронкой без моей просьбы xddicant.
- !queue - отображает все очереди | !queue GROUP - отображает очередь определённой группы.
- !skip - останавливает звук на общем канале (0) | !skip GROUP - остановить текущий звук на группе GROUP. MODERATOR ONLY
- !clear GROUP - полностью очищает очередь группы. MODERATOR ONLY
- !volume <0-100> - меняет громкость канала группы SONG. Потом придумаю настройку групп. MODERATOR ONLY
